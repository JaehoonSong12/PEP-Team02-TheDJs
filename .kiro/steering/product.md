# Product Overview

**Todo Management Application** — a full-stack web app that lets users register, authenticate, and manage hierarchical todo tasks with nested subtasks.

## Core Features
- User registration and JWT-based authentication (login/logout)
- CRUD operations for todo tasks (owned per user)
- CRUD operations for subtasks nested under a parent task
- Ownership enforcement: users can only access their own data
- Cascade delete: removing a task also removes its subtasks

## User Stories
1. As a new user, I can register an account to start tracking todo tasks.
2. As a user, I can log in and out to securely access my items.
3. As a user, I can create, edit, and delete todo items.
4. As a user, I can create, edit, and delete subtasks to organize my todos.
