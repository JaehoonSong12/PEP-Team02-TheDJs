\subsubsection{Entity Relationship Diagram}
Figure~\ref{fig:erd} shows a compact \textbf{Crow's Foot} (IE) style ERD.
Each \texttt{task} references exactly one \texttt{user} (\texttt{user\_id});
each \texttt{user} owns zero or more \texttt{tasks}.
Each \texttt{subtask} references exactly one \texttt{task} (\texttt{task\_id});
each \texttt{task} has zero or more \texttt{subtasks}.

\subsubsection{Integrity Rules}
\begin{itemize}
  \item \texttt{users.username} must be unique.
  \item Every \texttt{task} must reference a valid \texttt{user\_id}.
  \item Every \texttt{subtask} must reference a valid \texttt{task\_id}.
  \item Deleting a task cascade-deletes its subtasks (enforced in service layer).
\end{itemize}
