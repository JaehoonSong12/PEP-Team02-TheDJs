\subsubsection{GitHub Workflow}
\begin{enumerate}
  \item Create a branch from \texttt{main} for the work item (discussion notes or documentation).
  \item Commit changes locally; open a pull request when ready for review.
  \item A teammate reviews and approves; merge into \texttt{main}.
  \item Open or update a GitHub Issue for any remaining task IDs; link the issue to the pull request where applicable.
\end{enumerate}
